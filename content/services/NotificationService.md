# NotificationService

Sends alert notifications to players.

## Overview

A simple service for sending alert notifications to individual players or all players at once.

## Methods

### Alert

```luau
function Notification:Alert(player : Player, alertDetails : AlertDetails)
```

---

### AlertAll

```luau
function Notification:AlertAll(alertDetails : AlertDetails)
```

---

### AlertFor

```luau
function Notification:AlertFor(players : {Player}, alertDetails : AlertDetails)
```

---

### Init

```luau
function Notification:Init()
```
